# Weather App 🌦️

A desktop weather application built using Python and PyQt6.

This app fetches real-time weather data using an external weather API and displays it through a graphical user interface.

---

## 🚀 Features

- Real-time weather data retrieval
- Clean PyQt6 GUI
- Secure API key handling using environment variables
- Error handling for invalid input

---

## 🛠 Technologies Used

- Python
- PyQt6
- Requests
- python-dotenv

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/QuasiXD/weather_app
cd weather_app
```

### 2️⃣ Install required packages

```bash
pip install PyQt6 requests python-dotenv
```

### 3️⃣ Create a `.env` file

Create a file named `.env` in the project root and add:

```
api_key=your_api_key_here
```

⚠️ Do not upload your `.env` file to GitHub.

---

## ▶️ Run the Application

```bash
python main.py
```
